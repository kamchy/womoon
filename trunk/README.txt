Womoon 1.0
==========
Kamila Chyla <kamila.chyla@gmail.com>

Womoon jest prostym programem przeznaczonym do edycji diagramów
płodności, wykreślanych na podstawie codziennych obserwacji
(metoda objawowo-termiczna). 

Instalacja
----------

Aby zainstalować Womoon, należy z linii poleceń wprowadzić polecenie:
------------------------
python setup.py install
------------------------
